# zh-tools 工具包实现计划

> **目标:** 创建 `zh-tools` npm 工具包，提供上传、下载、日期格式化、图片处理等公用方法，使用 Vite 构建，支持 `npm install zh-tools` 安装使用。

**架构:** 使用 Vite library 模式构建，TypeScript 编写，输出 ESM + CJS 双格式。每个模块独立导出，支持 tree-shaking。

**技术栈:** Vite 5+, TypeScript 5+, Node.js 18+

---

### 文件结构

```
zh-tools/
├── package.json
├── vite.config.ts
├── tsconfig.json
├── tsconfig.build.json
├── README.md
├── src/
│   ├── index.ts          # 统一导出入口
│   ├── upload.ts         # 上传工具方法
│   ├── download.ts       # 下载工具方法
│   ├── date.ts           # 日期格式化工具
│   ├── image.ts          # 图片处理工具
│   └── types.ts          # 共享类型定义
```
